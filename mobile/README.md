# 📱 Versão Mobile — React Native

## Status: Planejamento

Esta pasta conterá a versão mobile do Vendedor IA para Android e iOS.

### Stack planejada:
- **React Native + Expo** (framework)
- **API compartilhada** (`src/services/dashboard-api.js`)
- **Auto-update OTA** (Expo Updates)
- **Deep linking** (`vendedoria://lead/123`)

### Timeline estimado:
- **Início**: Após estabilização da versão desktop
- **Duração**: 3-4 semanas
- **Release beta**: Q2 2026

### Requisitos:
- Node.js 18+
- Expo CLI
- Android Studio (para Android)
- Xcode (para iOS, apenas macOS)
